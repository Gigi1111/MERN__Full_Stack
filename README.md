# MERN__Full_Stack

This is a practice MERN full stack project using the following tools:


Frontend:
bootstrap,
reactstrap,
react-transition-group,
redux, 
thunk, 
axios, 
bcryptjs

Backend:
express, 
concurrently, 
nodemon

DB:
mongoose 


# Deployment
$npm install
Replace the mongoURI in config with your own mongoURI
