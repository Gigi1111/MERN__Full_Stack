# MERN__Full_Stack
main file: server.js

This is a practice MERN full stack project using the following tools:


Frontend:
react, 
bootstrap,
reactstrap,
react-transition-group,
redux, 
thunk, 
axios, 
bcryptjs

Backend:
nodeJS,
express, 
concurrently, 
nodemon

DB:
mongoDB
mongoose 


# Deployment
$npm install
Replace the mongoURI in config with your own mongoURI
